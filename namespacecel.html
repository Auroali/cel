void *  ptr 
)
inline

Used to communicate with the engine

Available signals are:
CEL_SIG_CAM_REQ: Takes a cel::camera** in ptr, will set to the current active camera
CEL_SIG_RENDER_PARAMS: Takes an uint64_t* in ptr, will set the render flags to this value. Only works in cel::project::init

Parameters
sigThe signal ID, valid IDs can be found in the cel.h header
ptrAdditional data to send with the signal (e.g. for a camera request, this would be a cel::camera double pointer)
